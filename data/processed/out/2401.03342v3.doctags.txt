<section_header_level_1><loc_226><loc_150><loc_273><loc_157>ABSTRACT</section_header_level_1>
<text><loc_88><loc_160><loc_413><loc_202>In these (not-completed) notes, we study the Hartogs extension phenomenon for holomorphic sections of holomorphic vector bundles over complex analytic varieties. Namely, we study properties of the Hartogs extension phenomenon with respect to the open embeddings, proper maps, compactifications, relations with the compact supports first cohomology, and the Lefschetz type property for sections of sheaves. As an application, we get a convex-geometric criterion of the Hartogs phenomenon for complex almost homogeneous algebraic G-varieties, where G is a semiabelian variety.</text>

<section_header_level_1><loc_59><loc_47><loc_126><loc_53>1 Introduction</section_header_level_1>
<text><loc_59><loc_63><loc_440><loc_77>The classical Hartogs extension theorem states that for every domain W ⊂ C n ( n > 1) and a compact set K ⊂ W such that W \ K is a connected set, the restriction homomorphism Γ( W, O ) → Γ( W \ K, O ) is an isomorphism.</text>
<text><loc_59><loc_80><loc_442><loc_101>A natural question arises: is this true for general complex analytic spaces? This phenomenon has been extensively studied in many situations, including Stein manifolds and spaces, ( n -1) -complete normal complex spaces, and so on (see, for instance, [24, 1, 6, 25, 34, 36, 37, 32, 16, 31, 40]).</text>
<text><loc_59><loc_104><loc_441><loc_118>In this paper, we use classical homological algebra methods (see, for instance, [35, 23]). For instance, we often use [23, Propositions 1.5.6, 1.8.7, 1.8.8.], facts from [23, Chapter II], and the diagram chasing.</text>
<text><loc_59><loc_122><loc_443><loc_198>In Section 3, we introduce the notion of the Hartogs sheaves (Definition 3.1). It is convenient to give this in more general settings. Using the Andreotti-Hill trick [1, corollary 4.3], we get for the case of normal complex analytic varieties that the finiteness of H 1 c ( X, O X ) implies that O X is Hartogs (Proposition 3.1). For any locally free O X -module of finite rank F , we obtain that the vanishing of H 1 c ( X, F ) implies that F is Hartogs (Lemma 3.2). Note that the Andreotti-Hill trick does not apply to arbitrary locally free O X -modules of finite rank. So, we have a natural question: is it true that the condition dim C H 1 c ( X, F ) < ∞ implies that F is Hartogs? This problem can be reduced to the case of holomorphic line bundles associated with effective Cartier divisors (Section 3.3). We solved this problem (Theorem 3.3) for the case of complex analytic manifolds X with the following additional assumption: X \ K admits non-constant holomorphic functions for a sufficiently large compact set K ⊂ X , and each irreducible divisor of X with compact support is F -removable (Definition 3.2). For example, each exceptional irreducible divisor is F -removable for any locally free O X -module of finite rank F (Example 3.6).</text>
<text><loc_58><loc_201><loc_442><loc_244>With some additional conditions, we may prove that the sheaf is Hartogs if and only if it is Hartogs over each domain (Corollary 3.4, Proposition 3.2). If a complex analytic variety has only one topological end, then we obtain the necessary and sufficient conditions of the Hartogs property (Theorem 3.1, Theorem 3.2). For example, each locally free O X -module of finite rank over a cohomological ( n -1) -complete complex manifold is Hartogs (Example 3.2), and for any domain X with connected boundary of a complex manifold X ′ with H 1 ( X ′ , O X ′ ) = 0 , we have that H 1 c ( X, O X ) = 0 implies that O X is Hartogs (Example 3.4).</text>
<text><loc_59><loc_246><loc_442><loc_294>The Hartogs property has a nice behavior with respect to the proper surjective holomorphic maps (Propositions 3.3, 3.4, 3.5). In particular, if X is a holomorphically convex noncompact normal complex analytic variety, R : X → Y is the corresponding Remmert reduction, and F ∈ V ect ( O X ) , then F is Hartogs if and only if dim Y > 1 (Corollary 3.1). More examples of noncompact holomorphically convex varieties can be obtained the following way: let X ′ be a normal complex projective variety, D be an effective basepoint-free Cartier divisor with connected support Supp( D ) , then X ′ \ Supp( D ) is holomorphically convex; moreover, the Remmert reduction induced by the holomorphic map associated with the complete linear system | D | (Example 3.5).</text>
<text><loc_59><loc_297><loc_443><loc_387>In Section 4, we consider very specific open embeddings. Namely, we consider ( b, σ ) -compactified pairs (Definition 4.2). Roughly speaking, the pair ( X, F ) (here X is a complex analytic variety, F is a sheaf of C -vector spaces) is ( b, σ ) -compactified if X admits a nice compactification X ′ such that X ′ \ X is a proper analytic set which has only b connected components, and the sheaf F extends to a sheaf F ′ over X ′ such that dim C H 1 ( X ′ , F ′ ) = σ . For pairs of the form ( X, O X ) , we have some facts on numbers ( b, σ ) . First, by the Nagata theorem, each complex algebraic variety admits a compactification in the category of complex algebraic varieties. It follows that each complex algebraic variety is ( b, σ ) -compactifiable for some ( b, σ ) . Now if X is a complex algebraic manifold (i.e., nonsingular complex algebraic variety), then the pair ( b, σ ) does not depend on any compactification in the category of complex analytic manifolds. Namely, b is exactly the number of topological ends of X , and σ is exactly the dimension of the Albanese variety of X (Proposition 4.1). In the case of almost homogeneous algebraic G -manifolds, σ is exactly the dimension of the Albanese variety of the open G -orbit (Remark 4.5). If G is a connected complex linear algebraic group, then σ = 0 , and the Sumihiro theorem implies that each complex algebraic G -manifold admits a G -equivariant compactification; hence X is ( b, 0) -compactifiable if and only if X has only b topological ends (Remark 4.5).</text>
<text><loc_59><loc_390><loc_441><loc_418>In Section 5, we study the Hartogs property for (1 , σ ) -compactified pairs (Theorem 5.1) and its relation with the Lefschetz property (Corollary 5.1, Remark 5.1). If the analytic set X ′ \ X is the support of a nef divisor, then we obtain a characterization of the Hartogs property in terms of the Iitaka and numerical dimensions of this divisor (Corollary 5.2).</text>
<text><loc_59><loc_421><loc_441><loc_456>In Section 6, we consider the case of almost homogeneous algebraic G -varieties and an example. We describe the space of germs of holomorphic functions at infinity analytic set (more precisely, the space Γ( Z, O X ′ | Z ) , where Z = X ′ \ X for some G -equivariant compactification X ′ of X ). If G is a semiabelian variety (i.e., G is an extension of an abelian variety by an algebraic torus), then we describe the space Γ( Z, O X ′ | Z ) in a more precise way. In particular, this implies the convex-geometric criterion of the Hartogs phenomenon (Theorem 6.1).</text>

<section_header_level_1><loc_59><loc_142><loc_193><loc_148>4.3 On ( b, σ ) -compactifiable varieties</section_header_level_1>
<text><loc_59><loc_154><loc_381><loc_161>In this subsection, we give any remarks on numbers ( b, σ ) for the categories B as in Definition 4.1.</text>
<text><loc_59><loc_164><loc_441><loc_199>Note that the number b is related to the topological ends of X (see [19, 18] or Remark 3.2 for more details about topological ends). Suppose X has e ( X ) topological ends. In general, b ≤ e ( X ) , but in the category An norm we have equality b = e ( X ) ; in fact, the Riemann extension theorem implies that if X is a connected normal space, then for each thin set A in X the space X \ A is connected ([9, Chapter 1, §13]). It follows that the number b does not depend on compactifications of X in An norm .</text>
<text><loc_58><loc_202><loc_442><loc_223>In the category An sm , the number σ is a bimeromorphic invariant. Namely, if X is ( b, σ ′ ) -compactified by X ↪ → X ′ , ( b, σ ′′ ) -compactified by X ↪ → X ′′ in the category An sm , and X ′ and X ′′ are bimeromorphically equivalent, then σ ′ = σ ′′ (see, for instance, [39, Corollary 1.4]).</text>
<text><loc_59><loc_226><loc_443><loc_248>The Nagata theorem [22] implies that each normal complex algebraic variety admits a compactification in the category Al a norm . By the resolution of singularities theorem, it follows that each complex algebraic manifold admits a compactification in the category Al a sm .</text>
<text><loc_59><loc_250><loc_441><loc_271>In the category Al a sm , the number σ coincides with the dimension of the Albanese variety (see [14] for more details about Albanese varieties). Recall that if X ′ is a compact complex algebraic manifold and Alb( X ′ ) is the Albanese variety of X ′ , then</text>
<formula><loc_152><loc_275><loc_348><loc_283>\dim _ { \mathbb { C } } A l b ( X ^ { \prime } ) = \dim _ { \mathbb { C } } \Gamma ( X ^ { \prime } , \Omega _ { X ^ { \prime } } ^ { 1 } ) = \dim _ { \mathbb { C } } H ^ { 1 } ( X ^ { \prime } , \mathcal { O } _ { X ^ { \prime } } ) .</formula>
<text><loc_59><loc_289><loc_442><loc_303>Now if X is ( b, σ ) -compactified by X ↪ → X ′ in Al a sm , then σ = dim C Alb( X ′ ) . Moreover, since the Albanese variety is a birational invariant, it follows that σ = dim C Alb( X ) .</text>
<text><loc_59><loc_306><loc_443><loc_320>By the remarks above, it follows that the numbers ( b, σ ) do not depend on any compactification in the category Al a sm . Namely, we obtain the following proposition:</text>
<text><loc_59><loc_323><loc_441><loc_337>Proposition 4.1. A complex algebraic manifold is ( b, σ ) -compactifiable in Al a sm if and only if X has b topological ends and dim C Alb( X ) = σ .</text>
<text><loc_59><loc_340><loc_443><loc_375>Remark 4.3. By the Sumihiro theorem [21, Theorem 3], it follows that if G is a connected complex linear algebraic group, then each noncompact normal complex algebraic G -variety admits a compactification in the category Al a G,norm (i.e., a G -equivariant compactification). By Brion's results [29], it follows that if G is an arbitrary connected complex algebraic group, then each noncompact normal quasiprojective complex algebraic G -variety admits a compactification in the category Al a G,norm .</text>
