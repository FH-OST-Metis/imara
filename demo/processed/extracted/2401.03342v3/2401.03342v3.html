HARTOGS AND OPEN EMBEDDINGS, PROPER MAPS, COMPACTIFICATIONS, COHOMOLOGIES

S. Feklistov

ABSTRACT

In these (not-completed) notes, we study the Hartogs extension phenomenon for holomorphic sections of holomorphic vector bundles over complex analytic varieties. Namely, we study properties of the Hartogs extension phenomenon with respect to the open embeddings, proper maps, compactifications, relations with the compact supports first cohomology, and the Lefschetz type property for sections of sheaves. As an application, we get a convex-geometric criterion of the Hartogs phenomenon for complex almost homogeneous algebraic G-varieties, where G is a semiabelian variety.

1 Introduction

The classical Hartogs extension theorem states that for every domain W ⊂ C n ( n > 1) and a compact set K ⊂ W such that W \ K is a connected set, the restriction homomorphism Γ( W, O ) → Γ( W \ K, O ) is an isomorphism.

A natural question arises: is this true for general complex analytic spaces? This phenomenon has been extensively studied in many situations, including Stein manifolds and spaces, ( n -1) -complete normal complex spaces, and so on (see, for instance, [24, 1, 6, 25, 34, 36, 37, 32, 16, 31, 40]).

In this paper, we use classical homological algebra methods (see, for instance, [35, 23]). For instance, we often use [23, Propositions 1.5.6, 1.8.7, 1.8.8.], facts from [23, Chapter II], and the diagram chasing.

In Section 3, we introduce the notion of the Hartogs sheaves (Definition 3.1). It is convenient to give this in more general settings. Using the Andreotti-Hill trick [1, corollary 4.3], we get for the case of normal complex analytic varieties that the finiteness of H 1 c ( X, O X ) implies that O X is Hartogs (Proposition 3.1). For any locally free O X -module of finite rank F , we obtain that the vanishing of H 1 c ( X, F ) implies that F is Hartogs (Lemma 3.2). Note that the Andreotti-Hill trick does not apply to arbitrary locally free O X -modules of finite rank. So, we have a natural question: is it true that the condition dim C H 1 c ( X, F ) < ∞ implies that F is Hartogs? This problem can be reduced to the case of holomorphic line bundles associated with effective Cartier divisors (Section 3.3). We solved this problem (Theorem 3.3) for the case of complex analytic manifolds X with the following additional assumption: X \ K admits non-constant holomorphic functions for a sufficiently large compact set K ⊂ X , and each irreducible divisor of X with compact support is F -removable (Definition 3.2). For example, each exceptional irreducible divisor is F -removable for any locally free O X -module of finite rank F (Example 3.6).

With some additional conditions, we may prove that the sheaf is Hartogs if and only if it is Hartogs over each domain (Corollary 3.4, Proposition 3.2). If a complex analytic variety has only one topological end, then we obtain the necessary and sufficient conditions of the Hartogs property (Theorem 3.1, Theorem 3.2). For example, each locally free O X -module of finite rank over a cohomological ( n -1) -complete complex manifold is Hartogs (Example 3.2), and for any domain X with connected boundary of a complex manifold X ′ with H 1 ( X ′ , O X ′ ) = 0 , we have that H 1 c ( X, O X ) = 0 implies that O X is Hartogs (Example 3.4).

The Hartogs property has a nice behavior with respect to the proper surjective holomorphic maps (Propositions 3.3, 3.4, 3.5). In particular, if X is a holomorphically convex noncompact normal complex analytic variety, R : X → Y is the corresponding Remmert reduction, and F ∈ V ect ( O X ) , then F is Hartogs if and only if dim Y > 1 (Corollary 3.1). More examples of noncompact holomorphically convex varieties can be obtained the following way: let X ′ be a normal complex projective variety, D be an effective basepoint-free Cartier divisor with connected support Supp( D ) , then X ′ \ Supp( D ) is holomorphically convex; moreover, the Remmert reduction induced by the holomorphic map associated with the complete linear system | D | (Example 3.5).

In Section 4, we consider very specific open embeddings. Namely, we consider ( b, σ ) -compactified pairs (Definition 4.2). Roughly speaking, the pair ( X, F ) (here X is a complex analytic variety, F is a sheaf of C -vector spaces) is ( b, σ ) -compactified if X admits a nice compactification X ′ such that X ′ \ X is a proper analytic set which has only b connected components, and the sheaf F extends to a sheaf F ′ over X ′ such that dim C H 1 ( X ′ , F ′ ) = σ . For pairs of the form ( X, O X ) , we have some facts on numbers ( b, σ ) . First, by the Nagata theorem, each complex algebraic variety admits a compactification in the category of complex algebraic varieties. It follows that each complex algebraic variety is ( b, σ ) -compactifiable for some ( b, σ ) . Now if X is a complex algebraic manifold (i.e., nonsingular complex algebraic variety), then the pair ( b, σ ) does not depend on any compactification in the category of complex analytic manifolds. Namely, b is exactly the number of topological ends of X , and σ is exactly the dimension of the Albanese variety of X (Proposition 4.1). In the case of almost homogeneous algebraic G -manifolds, σ is exactly the dimension of the Albanese variety of the open G -orbit (Remark 4.5). If G is a connected complex linear algebraic group, then σ = 0 , and the Sumihiro theorem implies that each complex algebraic G -manifold admits a G -equivariant compactification; hence X is ( b, 0) -compactifiable if and only if X has only b topological ends (Remark 4.5).

In Section 5, we study the Hartogs property for (1 , σ ) -compactified pairs (Theorem 5.1) and its relation with the Lefschetz property (Corollary 5.1, Remark 5.1). If the analytic set X ′ \ X is the support of a nef divisor, then we obtain a characterization of the Hartogs property in terms of the Iitaka and numerical dimensions of this divisor (Corollary 5.2).

In Section 6, we consider the case of almost homogeneous algebraic G -varieties and an example. We describe the space of germs of holomorphic functions at infinity analytic set (more precisely, the space Γ( Z, O X ′ | Z ) , where Z = X ′ \ X for some G -equivariant compactification X ′ of X ). If G is a semiabelian variety (i.e., G is an extension of an abelian variety by an algebraic torus), then we describe the space Γ( Z, O X ′ | Z ) in a more precise way. In particular, this implies the convex-geometric criterion of the Hartogs phenomenon (Theorem 6.1).
